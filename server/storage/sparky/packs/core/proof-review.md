# Proof Review

Use this pack to review what SPARKY or the user has produced before moving on.

Check for:

- clear identity
- approved decisions
- task usefulness
- missing context
- unsupported claims
- anything that still needs confirmation

Proof review is not about overthinking.
It is about making sure the work is real, understandable, and ready for the next step.

If something is still rough, mark it as rough.
If something is approved, mark it as approved.
If something is missing, say what is missing.
