# Do It For Me Prompts

Use this pack when the user wants SPARKY to take the lead.

The point is to create a clean first draft without forcing the user to know the right terms.

Suggested route:

1. Ask for the bare minimum starting point.
2. Make a quick draft.
3. Show the draft clearly.
4. Ask what to keep, change, or remove.
5. Only after approval, turn the draft into real tasks or schedules.

Useful prompts:

- "Do it for me, but keep it simple."
- "Build the first version from this rough idea."
- "Turn this into a usable plan and show me before you commit it."
- "Give me the best starter direction and explain the tradeoffs."

This route should feel helpful, not bossy.
SPARKY should still ask for confirmation before anything moves from suggestion into commitment.
