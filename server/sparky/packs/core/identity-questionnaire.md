# Identity Questionnaire

Use this pack when the user wants to build an identity, alter ego, project, brand, character, campaign, or business but does not know where to start.

Start with the minimum questions needed to find the shape of the idea.

Core questions:

1. What are we naming or building?
2. What feeling, lane, or world should it live in?
3. Who is it for?
4. What makes it different?
5. What must stay true no matter what?
6. What is still rough and undecided?

Helpful identity anchors:

- Name or working name.
- Voice or tone.
- Mission or purpose.
- Visual or cultural direction.
- Boundaries and do-not-do rules.

If the user is building an alter ego, treat it as a creative identity system unless they explicitly say it is a real-world identity matter.

Do not jump straight into execution.
Collect the smallest useful set of facts first, then show the shape back to the user for approval.
